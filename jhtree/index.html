jQuery Horizontal Tree plugin

Animated jQuery Horizontal Tree plugin for multi purposes scenarios with animation (Drag-Drop , Collaps , Expand)

Features

  • jQuery Plugin Based Architecture
  • Drage-Drop Nodes
  • Zoomer
  • Collaps & Expand
  • jQuery Ui Theme Supported
  • After Node Drop Event
  • Highlight Droped Node
  • Highlight Parent and children when mouse over
  • Animation Effects
