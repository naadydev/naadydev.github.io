jTLine

jQuery plugin, that create Horizontal Timeline,

Build upon the fantastic "codyhouse" article : https://codyhouse.co/gem/horizontal-timeline.


Features

  • jQuery Plugin Based Architecture
  • Auto Calculate Distance based on the Dates OR Time Variance
  • Auto Generate TimeLine Structure using Remote Ajax Call
  • Auto Generate TimeLine Structure by passing Json Object
  • Set Fixed Distance
  • Passing Pre-Defined Distance for each Object
  • Re-Design / Control / Formate the timeline ui objects
  • Point Clieck Event Handler
